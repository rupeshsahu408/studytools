## 1. Social Media Icons in User Profiles

When a user visits another user’s profile, the social links should be displayed properly using relevant icons.

* If a user has added a website link, the website icon should be shown.
* If they have added an Instagram link, the Instagram icon should be shown.
* If they have added a Reddit link, the Reddit icon should be shown.
* Similarly for other supported social platforms.

These icons should not just be static — they should be clickable:

* Clicking on the Instagram icon should redirect to the user’s Instagram profile
* Clicking on the Reddit icon should open their Reddit profile
* Website icon should open the provided website link

This should feel clean, modern, and intuitive like real social media platforms. The profile UI should dynamically detect which links exist and only show relevant icons.

## 2. Message Delete / Unsend Feature in Discussion Page

In the discussion (chat/community) page, users should be able to remove their own messages.

* This should work like Instagram DM:

  * The message should be completely removed from the chat for everyone
  * It should not just be hidden — it should be fully unsent/removed from the conversation

Add a delete option (e.g., three-dot menu or long press action) for each user’s own message.
